# Ханган Дейли

Статический новостной сайт для настольной ролевой игры «Аристократия».

## Публикация на GitHub Pages

1. Загрузить файлы из корня проекта в репозиторий.
2. В настройках GitHub Pages выбрать публикацию из ветки `main` или `master`, папка `/root`.
3. Главная страница: `index.html`.
4. Страница новости открывается через `article.html?id=идентификатор-новости`.

## Как добавлять новости

Новые материалы добавляются в массив `NEWS` в `scripts/news.js`.
Для карточки нужны `id`, `title`, `category`, `date`, `city`, `image`, `imageAlt`, `summary` и `body`.

Доступные категории: `sport`, `showbiz`, `politics`, `corporations`, `war`.
